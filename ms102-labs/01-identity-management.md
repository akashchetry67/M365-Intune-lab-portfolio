# MS102 Lab 1: Identity Management

Objective

Manage users and roles in Microsoft Entra ID.

Environment

Microsoft 365 E3 tenant

Steps

1. Open Entra Admin Center
2. Navigate to Users
3. Create new user
4. Assign roles

Roles Tested

Global Reader  
Exchange Administrator

Verification

Login using the new account.

Troubleshooting

If login fails:

Check MFA settings  
Verify license assignment
